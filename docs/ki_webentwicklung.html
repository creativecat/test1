Warum KI für Webentwicklung?

✅ Schnellere Entwicklung

Von der Idee zur lauffähigen Website in Minuten – nicht Stunden

✅ Keine Vorkenntnisse nötig

Einfach beschreiben, was die Website können soll – die KI liefert den Code

✅ Sofortige Ergebnisse

HTML, CSS3 und JavaScript – alles in einer Datei, bereit zum Testen

So funktioniert's im Workshop

1. Idee beschreiben

"Ich möchte eine Website, die X anzeigt und Y kann."

2. KI generiert Code

Mistral erstellt HTML, CSS3 und JavaScript – alles in einer Datei

3. Auf GitHub hochladen

Einfaches Deployment mit GitHub Pages – sofort online!

Datenschutz & Souveränität

Europäische Server, DSGVO-konform – Ihre Daten bleiben in Europa

Offene Modelle

Unabhängigkeit von US-Regulierungen
